# CONTRIBUTING

## Forudsætninger
- Git
- Java JDK 17
- IntelliJ IDEA (eller tilsvarende IDE)
- Maven (valgfrit – Maven Wrapper følger med projektet)

## Opsætning
1. Klon repoet
2. Åbn som Maven-projekt i IDE
3. Brug dev profil lokalt (se pplication-dev.properties)
4. Kør tests før du pusher:
   mvn test

## Branching og samarbejde
- Arbejd i feature branches
- Lav små commits med beskrivende commit-beskeder
- Brug Pull Requests når muligt
